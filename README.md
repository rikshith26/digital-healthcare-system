# HealthLab AI - Simple Digital Healthcare

HealthLab AI is a helpful website that makes it easy for you to take care of your health from home. You can book medical tests, talk to AI for health advice, and get your test reports online without any stress.

## What can you do with HealthLab AI?

*   **For You (as a Patient):**
    *   **Book Tests:** Pick a test and a time that works for you. A technician will come to your home to collect the sample.
    *   **Easy Reports:** No need to go to the lab. See and download your test results directly on your phone or computer.
    *   **Health History:** All your past tests are saved in one safe place so you can see them whenever you need.
    *   **AI Chat:** Ask our smart AI assistant about health topics anytime.

*   **For Lab Workers (as a Technician):**
    *   **Manage Bookings:** See new test requests from patients and accept them.
    *   **Upload Results:** Upload the patient's report as a PDF so they can see it instantly.

## How to get it running?

If you want to run this project on your own computer, follow these simple steps:

1.  **Install Python:** Make sure you have Python installed on your computer.
2.  **Get the code:** Copy all the project files into a folder.
3.  **Install requirements:** Open your terminal/command prompt and type:
    `pip install -r requirements.txt`
4.  **Database:** You will need a MongoDB database. Add your connection link in a file named `.env`.
5.  **Start:** Run the app by typing:
    `python app.py`
6.  **Open:** Finally, open your web browser and go to: `http://127.0.0.1:5000`
