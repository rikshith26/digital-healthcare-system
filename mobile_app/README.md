# HealthLab AI - Mobile Application

This folder is reserved for the mobile version of the HealthLab AI platform.

### Planned Architecture:
- **Framework**: Flutter or React Native
- **Connectivity**: Will connect to the central Flask API currently located in the `/web_app` folder.
- **Features**: 
  - Login/Register via API
  - View Diagnostic Reports
  - Book Lab Tests on the go
  - AI Health Assistant Chat

### Getting Started:
1. Ensure the web server is running via `run_app.bat`.
2. Initialize your mobile project in this directory.
3. Use the shared database via the central API endpoints.
